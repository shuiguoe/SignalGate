# SignalGate

**SignalGate 是一个“信号闸门”，不是信息系统。**

---

## 不可协商的硬原则（Non-Negotiable）

1. **默认沉默**
   - 系统不打断你，等价于：**什么都不要做**。

2. **唯一入口**
   - SignalGate **只有一个入口：命令行（CLI）**。
   - 禁止 UI、Web、后台常驻通知、弹窗等任何其他入口形式。

3. **禁止写死路径**
   - 代码中 **不得出现任何绝对路径**。
   - 所有路径只能来自：
     - 命令行参数
     - 环境变量
   - 否则视为架构违规。

4. **冷存优先**
   - 90–99% 的信息必须无声进入冷存。
   - 能打断你，永远是极端例外。

5. **零误导行动**
   - 系统唯一 KPI：
     > 由信息导致的错误行动次数 = **0**

---

## 本项目不是什么

- 不是资讯系统  
- 不是分析系统  
- 不是学习工具  
- 不是推荐引擎  

---

## 本项目是什么

一个**只在“必须行动”时才打断你的信号闸门**。

满足以下全部条件，才允许打断：

- 结构发生变化  
- 影响你已下注的对象  
- 需要你改变下一步行动  

除此以外，全部沉默。
