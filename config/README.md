# config/

本目录定义 **SignalGate 关心什么**。

这里只允许“配置”，不允许任何逻辑。

---

## 文件说明

- `bets.example.yaml`
  - 你已经下注的对象
  - 资产 / 项目 / 强制风险（税务、账户、合规）

- `sources.example.yaml`
  - 信息来源定义
  - 来源可信度分级（A / B / C）

- `rules.example.yaml`
  - 结构变化规则
  - 三问法阈值
  - 打断条件

---

## 强约束

- 本目录禁止出现任何可执行代码
- 禁止写观点、分析、解释性文字
- 所有字段必须**可被机器判断**
- 注释只允许解释“约束”，不允许解释“想法”

如果一个规则无法被清晰配置，
说明它不应该存在。
