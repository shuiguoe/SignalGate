# signalgate/

这里是 SignalGate 的**核心执行包**。

所有可执行逻辑只能存在于此目录。

---

## 架构硬规则

1. **唯一执行入口**
   - 只能由 `__main__.py` 启动系统。

2. **导入必须无副作用**
   - import 任何模块时，不得：
     - 读写数据
     - 执行收集
     - 产生输出
     - 触发打断

3. **默认沉默**
   - 模块只返回数据
   - 不允许 print / notify / log 到终端

4. **不确定即冷存**
   - 任何证据不足的判断，必须进入冷存

---

## 模块职责划分（不可越界）

- `core.py`      ：流程编排，只负责调用
- `paths.py`     ：路径解析（CLI / 环境变量）
- `models.py`    ：数据结构定义
- `ingress.py`   ：收集与规范化
- `decision.py`  ：三问法判断逻辑
- `gate.py`      ：限流 / 熔断
- `interrupt.py` ：打断消息构建（极简）
- `audit.py`     ：审计记录与回看

任何模块越界，视为架构失败。
